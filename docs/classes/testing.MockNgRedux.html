Class MockNgRedux<State>

This is the public of @angular-redux2/store. It wraps the global redux store and adds a few others add on methods. It's what you'll inject into your Angular application as a service.

Convenience mock to make it easier to control selector behaviour in unit tests.

getSelectorStub

  • getSelectorStub<SelectedState>(selector?: Selector<State, SelectedState>, comparator?: Comparator): Subject<SelectedState>
    • Defined in projects/angular-redux2/store/testing/services/ng-redux.service.mock.ts:152

    Returns a subject that's connected to any observable returned by the given selector. You can use this subject to pump values into your components or services under test; when they call .select or @select in the context of a unit test, MockNgRedux will give them the values you pushed onto your stub.

      it('select ticket price data from the substore', async(() => {
        const mockSubStore = MockNgRedux.getSubStore(
          ['WALLABIES', 'items', 'id1']);

        const selectorStub = mockSubStore.getSelectorStub('ticketPrice');
        selectorStub.next(2);
        selectorStub.complete();

        animalComponent.ticketPrice$
          .subscribe(
            ticketPrice => expect(ticketPrice).toEqual(2));
      }));

getSubStore

  • getSubStore<SubState>(...pathSelectors: PathSelector[]): MockNgRedux<any>
    • Defined in projects/angular-redux2/store/testing/services/ng-redux.service.mock.ts:180

    Returns a mock substore that allows you to set up selectorStubs for any 'fractal' stores your app creates with NgRedux.configureSubStore.

    If your app creates deeply nested sub stores from other sub store, pass the chain of pathSelectors in as ordered arguments to mock the nested sub store out.


      it('select name data from the substore', async(() => {
        const mockSubStore = MockNgRedux.getSubStore(['WALLABIES', 'items', 'id1']);

        const selectorStub = mockSubStore.getSelectorStub('name');
        selectorStub.next('Wilbert');
        selectorStub.complete();

        animalComponent.name$
          .subscribe(
            name => expect(name).toEqual('Wilbert'));
      }));
